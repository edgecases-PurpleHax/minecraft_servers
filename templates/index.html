Quick How to

  1. Open your bedrock client (the minecraft app)
  2. Click on the server tab
  3. Scroll to the bottom, click Add Server
  4. For the name field, enter whatever you want to call it
  5. For the Server Address enter minecraft.purplehax.com
  6. For the Port field, enter the port from the lists below that you want to connect to
  7. Follow the rules and have fun
